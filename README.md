# SecureVault – Python Password Manager

## Description
SecureVault is a simple password manager built in Python that demonstrates
cybersecurity concepts such as encryption, password hashing, and secure storage.

## Features
- Master password authentication
- Password encryption using Fernet (AES)
- Password strength checker
- Secure password vault
- Login attempt logging
- Password generator

## Technologies
- Python
- bcrypt
- cryptography
- JSON storage

## Project Structure
- main.py
- encryption.py
- database.py
- password_checker.py
- logs/login_logs.txt
- README.md
